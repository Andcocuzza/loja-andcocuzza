# Loja AndGuga

Projeto de e-commerce desenvolvido para praticar HTML, CSS e JavaScript, aplicando conceitos de estrutura semântica, responsividade e manipulação do DOM.

## Objetivo do Projeto

O objetivo deste projeto é evoluir conhecimentos em desenvolvimento frontend através da prática, criando uma loja virtual simples com navegação interativa e organização visual moderna.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

## Funcionalidades

* Slider interativo com navegação manual
* Layout organizado com Grid e Flexbox
* Página inicial com produtos em destaque
* Navegação entre páginas de produtos
* Estrutura HTML semântica
* Design responsivo em evolução

## Aprendizados Aplicados

Durante o desenvolvimento deste projeto foram praticados conceitos como:

* Organização de layout com Flexbox
* Criação de Grid de produtos
* Manipulação de elementos com JavaScript
* Estruturação semântica do HTML
* Versionamento com Git e GitHub
* Estilização moderna com CSS

## Melhorias Futuras

* Responsividade para dispositivos móveis
* Carrinho de compras
* Integração com backend
* Página de login
* Sistema de busca de produtos
* Animações e melhorias visuais
